重要事项

由于 AutoHotkey 1.0 和 AutoHotkey 1.1 的一些基本差异(尤其是 Unicode 和 x64 的可执行文件),为 AutoHotkey 1.0 编写的脚本可能无法正常运行在 AutoHotkey 1.1 上.关于已知的兼容性问题和解决方法的细节,请参阅下面:

  • 脚本兼容性 Unicode 和 ANSI 的比较,DllCall,NumPut/Get 以及其他变化
  • 脚本文件代码页 把您的文件保存为 UTF-8 编码!
