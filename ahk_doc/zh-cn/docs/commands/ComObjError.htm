ComObjError() [AHK_L 53+]

启用或禁用 COM 错误通告。

Enabled := ComObjError([Enable])

参数

Enable

布尔值 (真或假). 此参数是可选的.

Enabled

[v1.0.91+]: 保存调用函数前的实际设置.

一般说明

使用 COM 对象后, A_LastError 包含了由 COM 对象的 IDispatch::Invoke 函数返回的 HRESULT 码. 通过调用 ComObjError(false) 来禁用错误通知并检查 A_LastError 的值, 脚本可以实现自己的错误处理.

相关

ComObjCreate, ComObjGet, ComObjActive, ComObjConnect
